# Results Artifacts

This directory collects profiler exports, quantization summaries, and the rendered report generated by `make report`. Run the workflow end-to-end to refresh:

```
make quantize
make export_onnx
make convert_android
make android_profile
make report
```

Key files:
- `profile_summary.json` – host + Android latency/memory telemetry.
- `report.md` – formatted recap identical to the root `results.md`.
